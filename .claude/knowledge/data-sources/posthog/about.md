# PostHog

PostHog is an open-source product analytics platform.

## Key Features

- **Product Analytics** - Track user behavior, funnels, retention
- **Session Replay** - Watch user sessions
- **Feature Flags** - Roll out features gradually
- **A/B Testing** - Run experiments
- **Surveys** - Collect user feedback

## Pricing

- **Free tier**: 1M events/month, unlimited team members
- **Paid**: Starts at ~$0.00031/event after free tier
- **Self-hosted**: Free forever (you pay for infrastructure)

## Hosting Options

- **US Cloud** (`us.i.posthog.com`) - US data residency
- **EU Cloud** (`eu.i.posthog.com`) - EU data residency (GDPR)
- **Self-hosted** - Run on your own infrastructure

## MCP Integration

PostHog has an MCP server that lets Claude query your analytics directly:

- Ask questions in natural language
- Get insights without leaving your editor
- Create dashboards and funnels programmatically

This is why PostHog has "full support" in our kits - Claude can query it directly.
