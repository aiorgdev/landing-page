# Contributing

Thanks for your interest in the Landing Page Kit!

## Bug Reports & Feature Requests

**Issues are welcome.** Please open an issue if you:

- Found a bug
- Have a feature suggestion
- Have a question about how something works

## Pull Requests

**This repo is a read-only mirror.** Pull requests are not accepted and will be automatically closed.

The source of truth lives in the [aiorg.dev](https://aiorg.dev) monorepo. Changes are synced here on each release.

## Getting Help

- **Documentation:** https://aiorg.dev/docs/kits/landing-page
- **All kits:** https://aiorg.dev/kits
